Color del fondo:

El criterio color del fondo se encarga de identificar si el color de fondo de un recurso de aprendizaje es el correcto, para identificar esto se estableció que el color de fondo correcto sea el color blanco, por lo que si el recurso de aprendizaje tiene color blanco de fondo se obtendrá un porcentaje del 100%, de otra manera el criterio color del fondo tendrá un porcentaje del 0%.

Para obtener el valor del criterio color del fondo se utilizo la siguiente formula:

CFO: Si el color del fondo es igual ha color blanco (CB), entonces el valor del criterio color del fondo es igual al cien por ciento.

CB: Es el valor del color de fondo blanco (CB).


CFO: Si el color del fondo es igual a otros colores (OC), entonces el valor del criterio color del fondo es igual al cero por ciento.

OC: Es el valor de otros colores de fondo (OC).
