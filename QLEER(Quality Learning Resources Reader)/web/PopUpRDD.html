¡Sub-Atributo Disponibilidad!

Disponibilidad: Se define como el grado en el cual un servicio Web de aprendizaje puede ser utilizado en los contextos para los que es construido y en cualquier momento.

El criterio disponibilidad se define de acuerdo a la estabilidad del servicio Web de aprendizaje (SWA) como tal y en este caso sólo puede haber dos valores, es decir, si el SWA responde a la solicitud, el criterio disponibilidad tomará el valor de 1 y si el SWA no responde la solicitud, el criterio disponibilidad tomará el valor de 0, por lo tanto si hay respuesta D = 1 de otra manera D = 0 [1].
