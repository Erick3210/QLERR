Color de la fuente:

El sub-atributo de Color de la fuente se encarga de identificar el Color de la fuente (color de letra) correcto de un recurso de aprendizaje, para identificar esto se estableció que los colores de la fuente correctos son negro, azul, rojo, y verde, por lo que todos colores de la fuente correctos de un recurso de aprendizaje se promediarán para sacar el porcentaje del criterio color de la fuente.

Antes de calcular el valor del criterio color de la fuente, se debe calcular el valor de aquellos sub-atributos con una ponderacion menor a uno, estos son los sub-atributos color rojo y color verde.

Para obtener el valor del sub-atributo color rojo se utilizo la siguiente formula:

CR: Es el valor de las palabras en color rojo.


Para obtener el valor del sub-atributo color verde se utilizo la siguiente formula:

CV: Es el valor de las palabras en color verde.


Para obtener el valor del criterio color de la fuente se utilizo la siguiente formula:

CF: Color de fuente es igual a la adición de los valores de las palabras en color rojo (CR), color verde (CV), color negro (CN), y color azul (CA), entre el valor del total de palabras (TP).

CN: Es el valor de las palabras en color negro.

CA: Es el valor de las palabras en color azul.

CR: Es el valor de las palabras en color rojo.

CV: Es el valor de las palabras en color verde.

TP: Es el valor del total de palabras.

OC: Es el valor de las palabras en otros colores, el cual no se toma en cuenta para obtener el valor del criterio color de fuente.
