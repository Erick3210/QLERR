¡Sub-Atributo Usabilidad!

Usabilidad: Se define como el grado en que un producto puede ser utilizado por usuarios específicos para lograr objetivos concretos, eficacia y satisfacción en un determinado contexto de utilización.

En este caso como se define como la capacidad del servicio Web de Aprendizaje para que el contenido del recurso sea accesible y legible [1].

En otras palabras para que un recurso de aprendizaje sea usable, dicho recurso debe ser accesible y legible, esto se determina gracias a una serie de sub-atributos (Alineación, Tipo de letra, Color de fuente, Tamaño de fuente, Color de fondo) que contiene un recurso de aprendizaje dentro de un Servicio Web de Aprendizaje (SWA), el siguiente diagrama muestra una parte del modelo de calidad que representa la usabilidad de un recurso de aprendizaje y como obtenerla:
