Orientación del texto:

El sub-atributo de orientación del texto se encarga de identificar si el texto de un recurso de aprendizaje esta alineado, para identificar esto se estableció que los parrafos del texto de un recurso tengan una alineacion justificada, a esto se le denomino parrafos adentro, por lo que todos los parrafos adentro de un recurso de aprendizaje se promediarán para sacar el porcentaje del criterio de orientación del texto.

Para obtener el valor del sub-atributo orientación del texto se utilizo la siguiente formula:

OT: Orientación del texto es igual a la diferencia del valor de los párrafos adentro (PAD) entre el total de párrafos (TP).

PAD: Es el valor de los párrafos adentro.

TP: Total de párrafos.
